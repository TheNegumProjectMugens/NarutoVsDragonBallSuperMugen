Testamento

Esta es mi segunda version de mi primer personaje, SuperVegeta, ya que sinceramente lo quiero terminar antes de recibir mi jubilacion anticipada. Y ademas comienzo a sentir un extrano sentimiento de odio y rencor de niveles estratosfericos hacia este personaje incompleto. Como ya se ha terminado, ahora he alcanzado el nirvana, hasta que comienze con otro proyecto o decida lanzar una version 3.0.

Resta decir que el personaje es, 100% Open Source.
